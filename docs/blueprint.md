# **App Name**: SmartEdu Platform

## Core Features:

- User Authentication and Roles: Secure user authentication with role-based access control (director, supervisor, teacher, student, parent) using MySQL.
- Lesson Management: Create, edit, and manage lessons with title, content, video URL, PDF URL, subject, level, author, type, and status, connected to MySQL database.
- Exercise Management: Create and manage exercises linked to lessons, including questions, model answers, PDF links, and linguistic checks, all data in MySQL database.
- Student Submissions and AI Feedback: Allow students to submit answers to exercises, and provide AI-driven feedback on their submissions, stored in MySQL. An AI tool determines which specific feedback prompts are most appropriate based on a probabilistic calculation using information extracted from the user submission and the student profile.
- Internal Messaging System: Enable internal messaging between users with different roles (e.g., teacher to student, director to teacher), stored in MySQL.
- Student Progress Tracking: Track student progress in lessons, including completion date and score, stored in MySQL.
- User Profile Management: Enable users to manage their profiles, including updating personal information, avatar, and communication preferences, data stored in MySQL.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Very light blue (#E8EAF6), a muted version of the primary.
- Accent color: Purple (#7E57C2), to contrast and highlight key elements.
- Body and headline font: 'Inter', a sans-serif with a modern, machined, objective, neutral look; suitable for both headlines and body text
- Consistent and clear icons for navigation and actions.
- Clean and intuitive layout with a focus on user experience.
- Subtle transitions and animations for a smooth user experience.
